Aouracord

Discord customization mod

Download

Aouracord allows you to completely modify discord to your liking, using Themes and Plugins. I want to state that Aouracord is a modification of Vencord Web. I will be adding desktop support soon.

Download

Chrome: Download the CrossPilot extension from the Chrome Web Store, and download the file below. Open CrossPilot and upload the file. If the extension installs itself when clicking the button, stop following instructions. If the extension is blocked, go to extensions and enable it anyway.

Opera / OperaGX: On the Opera webstore, search "Aouracord" and install it.
